=============
akaihola-tools
=============

Collection of useful Python tools

Features
========

 * Command-line tools interface
 * GitHub repository cloning tool

Installation
============

.. code-block:: bash

    uv tool install akaihola-tools

or

.. code-block:: bash

    pip install akaihola-tools

Usage
=====

List available tools:

.. code-block:: bash

    tools --list

Run a specific tool:

.. code-block:: bash

    tools <tool_name> [arguments]
